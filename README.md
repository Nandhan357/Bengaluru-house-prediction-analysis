# 🏠 Bangalore House Price Prediction

## 📖 Overview

This project predicts house prices in Bangalore using Machine Learning regression models. It demonstrates the complete machine learning workflow, from data preprocessing and exploratory data analysis (EDA) to model training, evaluation, and prediction.

The project uses property features such as location, total square footage, number of bedrooms, bathrooms, and other relevant attributes to estimate house prices.

## 🎯 Objectives

* Clean and preprocess the dataset.
* Perform Exploratory Data Analysis (EDA).
* Encode categorical features.
* Train regression models to predict house prices.
* Evaluate model performance using regression metrics.
* Predict prices for new house data.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## 📊 Exploratory Data Analysis

The following analyses were performed:

* Distribution of house prices
* Correlation heatmap
* Boxplot for outlier detection

## 🤖 Machine Learning Models

* Linear Regression
* Decision Tree Regressor

## 📈 Evaluation Metrics

The models were evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score
